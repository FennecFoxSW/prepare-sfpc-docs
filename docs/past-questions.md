# SFPC 기출 문항

- 모든 문항은 배경이 같은 주된 문항 아래 세부 문항이 나뉘어 있는 형식.
- 모든 문항에 대해 문항 코드가 `0`으로 끝나면 서답형 문항, 그렇지 않으면 구현 문항. 서답형 문항은 CodeUp에 Special Judge로 등록되어 있음.
- 성공률은 작성 시점 기준.
